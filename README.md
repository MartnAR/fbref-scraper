# FBRef Scraper
--------
This repository contains Python functions and scripts capable of scraping data from [FBRef](https://fbref.com). 

## Description
--------
Using `seleniumbase` and `playwright`, these series of functions are capable of bypassing FBRef's Bot verification, accessing the website's different content sections
and allowing users to scrape data. 
Currently, the scraper is only able to obtain teams within the top 5 leagues in a given season, all the matches a team has played in a given season, and match reports
for a team in a single season. As this project expands, more functions and use cases will be added. 

## Dependencies
---------
* `python` 3.13.5
* `seleniumbase` 4.50.5
* `playwright` 1.61.0
* `pandas` 2.3.3
